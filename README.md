# 多功能 AI 应用后端

这是一个基于 Go 和 Eino 框架构建的多功能后端应用，整合了 AI 搜索、秒杀系统等核心功能。
1. 支持知识库优惠券秒杀功能，主要涉及组件 Redis , MySQL, RocketMQ。
2. 支持 AI agent, 使用eino搭建 AI agent 框架，基于 RAG 匹配知识库文档与用户进行交互。
3. 支持普通文档检索，基于 Elasticsearch 实现倒排索引搜索。


## 秒杀系统

高并发优惠券秒杀系统，支持大规模用户同时抢购优惠券，确保库存准确性和系统稳定性。

**核心特性：**
- **原子性库存扣减**：使用 Redis + Lua 脚本保证库存扣减的原子性，避免超卖问题
- **异步订单处理**：通过 RocketMQ 消息队列实现订单异步处理，提升系统吞吐量
- **自动补偿机制**：MQ 发送失败时需要进行补偿
- **分布式缓存**：利用 Redis 缓存热点数据，减轻数据库压力
- **高可用设计**：多层容错机制，确保系统稳定运行

**技术栈：**
- Redis：库存缓存与原子操作
- MySQL：持久化存储优惠券与订单数据
- RocketMQ：异步消息队列，削峰填谷
- Lua 脚本：保证 Redis 操作原子性

**业务流程：**
1. 用户发起秒杀请求
2. 系统通过 Lua 脚本原子性扣减 Redis 库存
3. 库存充足时创建订单并发送到 MQ
4. MQ 消费者异步处理订单，写入数据库
